# Python Portfolio

## Overview
This repository contains Python mini projects built to practice
automation, problem solving, and basic data handling using Python.

The goal of this portfolio is to demonstrate practical Python skills
through simple, real-world projects.

---

## Project: Expense Tracker

### Description
The Expense Tracker is a simple Python script that allows users to
record expenses and view them through a command-line interface.
It focuses on core Python concepts such as variables, loops,
conditionals, and user input.

---

### Features
- Add new expenses
- View recorded expenses
- Simple and easy command-line interaction
- Beginner-friendly Python logic
